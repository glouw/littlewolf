Littlewolf aims to be a very minimalistic software graphics
engine reminiscent to some of the early works of Carmack at id.
HOPE YOU PEOPLE LIKE IT
Thanks to all!

    make; ./littlewolf

Dependencies:

    SDL2-devel

Controls:

    move: W,A,S,D

    turn: H,L

    exit: END, ESCAPE
